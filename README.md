# FlowPublisher
FlowPublisher is a lightweight content publishing tool designed to manage and publish vertical video content to owned TikTok accounts using structured workflows.

The application focuses on assisting professional creators in organizing, scheduling, and publishing video content through user-initiated processes. All actions are performed only on accounts explicitly authorized by the user via TikTok’s official OAuth authentication.

## Purpose

FlowPublisher was created to support content management workflows, allowing creators to publish short-form vertical videos efficiently while maintaining full control over their accounts and content.

## Key Features

- Publish vertical video content to owned TikTok accounts
- Workflow-based publishing logic
- OAuth-based account authentication
- Content organization and management support

## Data Usage

FlowPublisher accesses only the data necessary to provide its publishing and management functionality. No data is shared with third parties, and access can be revoked at any time through TikTok account settings.
